# Cómo descargar los diagramas en PNG

Ya dejé los diagramas en formato Mermaid editable (`.mmd`) dentro de `diagramas_mmd/`.

## Archivos disponibles
- `diagramas_mmd/E-8.5.mmd`
- `diagramas_mmd/F-8.6.mmd`
- `diagramas_mmd/G-8.7.mmd`
- `diagramas_mmd/M-7.13.mmd`
- `diagramas_mmd/O-8.15.mmd`
- `diagramas_mmd/P-8.16.mmd`

## Opción A (recomendada): Mermaid Live Editor
1. Abre https://mermaid.live/
2. Copia el contenido de uno de los archivos `.mmd`.
3. Pégalo en el panel izquierdo del editor.
4. En **Actions** elige **Download PNG**.
5. Repite para cada archivo.

## Opción B: Visual Studio Code
1. Instala una extensión compatible con Mermaid (por ejemplo, Mermaid Markdown Syntax Highlighting + Mermaid preview).
2. Abre un archivo `.mmd`.
3. Previsualiza el diagrama.
4. Usa la opción de exportar/guardar imagen como PNG (según la extensión).

## Nota
Si quieres, en el siguiente paso también te puedo dejar una carpeta `diagramas_png/` ya generada automáticamente, cuando el entorno permita instalar `@mermaid-js/mermaid-cli`.
